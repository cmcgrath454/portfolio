Big O API

This tool can be used to analyze the Big O runtime complexity of a Java code snippet. Replace the code in the input box below and click analyze to see the result. Please note the following about the tool:

  • Recursion complexity analysis is not currently supported
  • For simplicity, this tool is meant for code snippets, not class or method declarations.
  • This tool attempts to catch and highlight unsupported cases in red and underlined. Any such code will not be calculated in the end result.
  • There is an abundance of possible ways to initialize, terminate, and update loops, so this tool sticks to covering the most common methods used.
  • Only shorthand assignment operators (+=, -=, *=, /=) for updating looping variables are currently supported
  • For runtime analysis, we assume that our input is int variable 'n'
