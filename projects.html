Java Runtime Complexity API

JavaScript | Jest | AWS Lambda + API Gateway

For my university capstone project, I worked with a team to create a runtime complexity learning tool. My role was to write the backend logic to analyze Java code runtime complexity. I created an API and hosted it through AWS. Other members of the group created a front-end that called the API to analyze user code input.

I wrote the API in JavaScript, used npm package manager, employed a Java syntax tree parsing package, and used Jest for testing. I created a simple interactive form for users of this site to test out the API with. The API is currently hosted on AWS Lambda with calls routed through AWS API Gateway.
